# Research Digest Prompt

You are generating a daily research digest for a technical researcher working across robotics, computer vision, reinforcement learning, surgical AI, and quantum computing.

For each item, output:

- Title
- Link
- Area
- 5-bullet technical summary
- Why it matters
- Possible implementation idea
- Relevance score from 1-5
- Confidence and caveats

Only include items that are technically meaningful. Avoid hype.
