# Power Automate Flow: Weekly Personal Review

## Trigger
Every Friday at 16:30.

## Inputs
- Outlook calendar events from the last 7 days
- Microsoft To Do / Planner tasks
- flagged emails
- OneDrive/SharePoint modified files

## Output
Send yourself a weekly review email with:

1. Completed commitments
2. Missed/at-risk tasks
3. Important meetings
4. Follow-ups needed
5. Next week priorities

## Safety
Read-only except for creating a draft planning task if enabled.
