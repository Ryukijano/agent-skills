# Canvas Design

## Design Philosophy Creation
- THE CRITICAL UNDERSTANDING: Every visual design starts with a philosophy
- HOW TO GENERATE A VISUAL PHILOSOPHY: Understand the subject, audience, and medium
- PHILOSOPHY EXAMPLES: Minimalism ("less is more"), Bold ("make a statement"), Organic ("nature-inspired")

## Essential Principles
- Balance: Visual weight distribution
- Hierarchy: Guide the eye through the design
- Contrast: Create emphasis through difference
- Unity: All elements should feel connected

## Deducing the Subtle Reference
- Look at the subject matter for visual cues
- Consider cultural and historical context
- Find the "feel" before the "look"

## Canvas Creation
- Use Python (PIL/Pillow, Cairo) or HTML/CSS
- Define canvas size based on output medium
- Build in layers: background, midground, foreground

## Final Step
- Export at 2x resolution for print quality
- Save both PNG (digital) and PDF (print) versions

## Multi-Page Option
- Maintain visual consistency across pages
- Vary layout while keeping the design system
