# Explore SOTA

Search arXiv and Semantic Scholar, build a triage queue, digest top papers.

Apply skills: `explore-sota`, `digest-paper`, `parallel-web-search`.

1. Frame the research question and evaluation criteria.
2. Search with `parallel-web-search` or Browser MCP; cite sources.
3. Triage: adopt / optional / reject per paper with repo fit.
4. Optionally run `digest-paper` on top hits.

For surgical MOT: pair with `/mot-browser-research` and GOT-JEPA guardrails.
