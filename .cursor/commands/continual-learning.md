# End-of-Session Handoff

1. `git status` — stage or note every change.
2. Commit meaningful work: `git commit -m "..."` or create a clear WIP branch.
3. Write or update `handoff-<slug>.md` with next first action.
4. Update any relevant skill or AGENTS note.
5. Run the project's smoke / verify one last time and record outcome.
6. Close the loop: no open uncommitted state that would surprise the next session.

Apply `continual-learning`, `git-branch-workflow`.
